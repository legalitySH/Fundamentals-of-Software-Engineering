Оценка проекта

Опыт персонала

Персонал данного проекта не имеет огромного опыта работы, однако данная лабораторная работа позволит персоналу получить необходимый опыт для получения нужных знаний и умений для работы в команде.Персонал владеет базовыми основами языка С++, С.

Размер проекта и тип проекта

Размер проекта является средний. Проект классифицируется по следующим типам:

  1. Тип проекта : краткосрочный
  2. По сложности : средней сложности
  3. По требованиям к качеству и способам его обеспечения: Бездефектный
  4. По совокупности проектов : Мультипроект
  5. По уровню участников : местный
  6. По характеру целевой задачи : бытовой
  7. Чрезвычайная ситуация : реинжиниринг
